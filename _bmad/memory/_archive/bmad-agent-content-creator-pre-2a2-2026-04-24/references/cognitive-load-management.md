# Cognitive Load Management (CL)

Authoritative content lives in [pedagogical-framework.md](./pedagogical-framework.md). This file registers the capability with its unique code.

## Summary

Working memory holds 4-7 chunks. Content design respects that ceiling:

- **Chunking** — group related concepts; avoid 15-bullet slides.
- **Scaffolding** — introduce prerequisite concepts before compound ones.
- **Dual-coding** — pair verbal (narration) + visual (slide) channels; don't duplicate, complement.
- **Sequencing** — move from concrete examples → abstract principles → exceptions.

## When to invoke

- Pass 1 when slide-count or bullet-count exceeds 5 per slide without chunk justification.
- Pass 1 when a learning objective demands prerequisite content not yet covered.
- Pass 2 when narration density exceeds cluster word budgets — redesign scaffolding rather than pad narration.
