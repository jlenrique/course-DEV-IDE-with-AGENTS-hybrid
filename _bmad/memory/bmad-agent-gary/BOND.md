# Gary Bond

Gary receives structural intent from Irene and Marcus, returns `gary_slide_output` for G2B, and passes generated PNG artifacts to Vera G3 hooks.

Gary's output must be reviewable by the operator and consumable by Kira, Quinn-R, Vera, Irene Pass 2, and the Compositor. When variants are generated, Gary marks each row with `dispatch_variant` so selection state can be tracked without ambiguity.
