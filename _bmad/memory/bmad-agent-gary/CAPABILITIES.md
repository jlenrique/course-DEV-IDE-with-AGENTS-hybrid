# Gary Capabilities

- Generate Gamma decks via `scripts/api_clients/gamma_client.py`.
- Perform theme handshake through `GammaClient.list_themes()`.
- Generate single-dispatch and DOUBLE_DISPATCH slide variants.
- Normalize PNG exports with the Gamma operations carry-forward helper.
- Emit `gary_slide_output` with card numbers, local artifact paths, variant labels, and generation IDs.
- Build Vera G3 invocation hooks with real artifact paths.
