## Canonical Pattern Assessment

Reusable helpers:

- `canonical_model_bytes(model: BaseModel)` is domain-agnostic.
- `compute_model_digest(model: BaseModel)` is domain-agnostic.
- The transport handlers delegate through one neutral `submit_verdict(...)` path.
- Section 02A-specific edit behavior is isolated in `apply_directive_edit(...)`, where downstream stories can replace only the domain-model edit/re-validation step.

Verdict payload equivalence:

- Focused tests assert CLI, HTTP, and MCP-stdio handlers emit identical `operator_verdict` payloads for the same directive and operator action.
- The deterministic parity timestamp defaults to the directive `composed_at`; callers can pass `submitted_at` explicitly for real transport submissions.
