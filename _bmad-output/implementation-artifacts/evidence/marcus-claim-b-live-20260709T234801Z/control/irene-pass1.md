# Irene Pass-1 Lesson Plan

A concise closing-and-bridge lesson that synthesizes Module 1, reframes innovation success around the leader rather than the idea, marks the learner's shift from reacting to redesigning, and previews Module 2's focus on leadership identity, communication, and inspiration. Cluster use is minimal and targeted: only the idea-versus-leadership claim is expanded because it contains multiple explanatory beats and high pedagogical weight. All other units remain keep-dense to preserve the integrity of the closing synthesis.
