## Standing watchpoints (this arc)

1. **No orthogonal-switch machinery.** `production_mode` is a DECLARATIVE type tag; the existing Studio dispatch fork (`_act.py:943`) stays as-is. Reject any compose-at-resolve / overlay engine (operator rejected "Approach 2").
2. **No Classic-key leakage onto Studio.** Discriminated union keeps Classic surface keys un-representable on a studio record (`extra="forbid"` on both the Studio arm and `studio_template`); Studio overrides must not ride a Classic record. Both directions enforced.
3. **`derived_from` is inert provenance.** Optional, Studio-scoped, string. The validator may only check the referent exists. **No resolver path may ever read/consume it** — that would be the orthogonal engine in disguise.
4. **Overrides are optional; absence ≠ empty.** Absent `theme_id`/`image_model` ⇒ the dispatch kwarg is OMITTED (never `themeId=""` / `imageOptions={"model": None}`). No global-default back-fill (e.g. recraft-v3 for everyone).
5. **Wire receipt is the only non-tautological oracle.** Every test asserts against the NEXT layer's observed input / captured outbound payload — never a same-test input echoed back. A test that passes with the client uncalled is worthless. The load-bearing test: dispatch → recording client captures `themeId` + `imageOptions={"model": …}`.
6. **No-silent-drop (bundle-carrier invariant).** A studio record with overrides must never silently lose them end-to-end resolver→dispatch; a layer that can't carry them fails loud.
7. **Single image-model enum registry.** `image_model` enum-validates against the ONE existing catalog enum (`IMAGE_MODEL_VALUES` surface) — no second hand-copied list. Drift test: validator accepted set == catalog enum. `theme_id` is an opaque Gamma id → string-validate only, no enum.
8. **`image_options` is exactly `{"model": …}`.** Never forward `source`/`stylePreset` (from-template 400s). Negative wire assertion required.
9. **English-only steer travels inside the prompt.** The steer must be composed INSIDE `_STUDIO_LOCK_WRAPPER` so it reaches the model (Studio bakes text into the image; "English-only", not "no text"). Test asserts outbound prompt string contains it.
10. **Source-detail → Gamma conveyance never degrades** (protected invariant). Base-layer overrides compose UNDER source-derived instructions/keywords/prompts; source wins on conflict.
11. **Live gate integrity.** 2 cards; receipt = primary oracle, generative render = corroboration only (no pixel-match gate); determinism asserted on the PAYLOAD not the render; first-run-stands, no retry-to-green; receipt pasted into Completion Notes.
12. **Scope discipline.** Promotion is OUT this session (candidate only; operator-eye gate; sibling-parity — can't promote above the still-candidate Classic parent). Hard step-1→step-2 checkpoint: plumbing green before any paid render. Transformer skill stays deferred (built against a proven hand-mapping, not in the abstract). Do not absorb baseline residuals (ruff app/, C3 lint-imports, contracts-tree 2) into this arc.
