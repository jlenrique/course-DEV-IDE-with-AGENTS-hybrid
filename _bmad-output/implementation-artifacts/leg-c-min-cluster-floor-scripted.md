## Why (product value / guardrail)
Give a styleguide a way to declare a **style-identity** property that is genuinely NOT expressible declaratively: "this style reads as a multi-beat narrative walk; below N beats it collapses to a stub deck." That intent is honored by feeding Irene's Pass-1 clustering a **minimum cluster floor**, improving VO **follow-along** on multi-beat styles (dense source chunked so narration can track it — [[project_chunking_via_clustering_followalong]]). PRODUCT-VALID: the floor must earn its place in follow-along/perception terms, **never** to make a proofing run produce a particular deck ([[feedback_spoc_is_goal_not_concierge_proofing_runs]]). **The floor proposes; the non-waivable 07G gate disposes.**

## Acceptance criteria (RED-first; DUAL-GATE + 07G non-waivable)
**Contract / structural (Murat):**
- **AC#1 [scripted enum + accessor]** `min_cluster_floor` is a registered `scripted_enum` class; the scripted-only accessor returns the typed positive-int; an unregistered `scripted` class → RED `gamma.scripted.unknown-class`; a missing/empty `rationale` → RED. A synthetic non-int/negative value → RED.
- **AC#2 [leak guard]** `scripted.*` NEVER appears in the resolved `gamma_settings[]` surface — a test asserts the accessor and the Gary resolver are non-overlapping projections (a `min_cluster_floor` in a styleguide does not reach any Gamma payload key).
- **AC#3 [threading — early-bound]** with a directive binding a styleguide whose `scripted.min_cluster_floor = N`, `_runner_payload_for_specialist("irene-pass1", …)` returns `{"min_cluster_floor": N}`; the value arrives in Irene's `envelope_payload` at `_act_pass_1`. Absent styleguide/scripted → no key (no crash, no `None`-leak).

**Honoring / safety (Irene + Amelia):**
- **AC#4 [split-only, byte-identity — the 07G guard in unit form]** feed a known Pass-1 output (M clusters), apply floor N > M: assert (a) result has ≥N clusters AND (b) the **flattened text sequence is byte-identical to pre-floor** (subdivide-only; no merge/reorder/reassign).
- **AC#5 [figure↔narration atomicity]** a cluster containing a figure + its explaining narration, under floor pressure, is NEVER split across that bond (explicit pre-check test); the bond survives intact.
- **AC#6 [soft floor / content veto]** floor N on content whose legitimate seams top out at M < N → emit `styleguide-vs-content mismatch` fail-loud, and DO NOT over-fragment (no forced bad split). The mismatch is the failure surface, not a manufactured split.
- **AC#7 [dead-config detector — narrow, IN-SCOPE]** a `min_cluster_floor` set on a bound styleguide but never consulted at Pass-1 → **fail loud** (discriminating pair: consulted → quiet; set-but-never-read → raise). An unknown `scripted` key → RED (covered by AC#1; assert it's not silently tolerated as "dead").
- **AC#8 [learned-store-stays-empty pin]** Leg-C introduces ZERO learned entries; the Leg-B2 learned store + manifest remain empty; the only floor in play came from the documented `scripted` class.

**Live proof (no mocks, real Irene Pass-1, first-run-stands):**
- **AC#9 [differential-effect-fired pair]** same corpus + seed, floor-off (control K) vs floor-on (treatment T); arbiter = cluster count read from the **on-disk Pass-1 artifact** (never a driver's inline claim). **Anti-vacuous: `count_K < N ≤ count_T`** — the control MUST cluster below the floor. (If no real corpus clusters below N with legitimate seams to reach N → the effect is unfalsifiable → BLOCK.)
- **AC#10 [determinism pinned]** run K twice → `count_K == count_K'` before attributing the T−K delta to the floor.
- **AC#11 [07G with teeth]** the read-path-match gate returns a real PASS **verdict artifact** on the floored deck T; a **teeth-witness** (deliberately desynced floored deck — one cluster's VO shuffled against its on-screen text) is FLAGGED RED by 07G. ("07G didn't error" is rejected as vacuous.)
- **AC#12 [figure-bond-survives, live]** a real cluster containing a figure + its reading, floor applied, demonstrated intact on-disk.
- **AC#13 [adversarial floor-too-high → honest refusal — Irene, non-waivable]** a floor set above what the corpus can honestly support emits the mismatch signal and **declines to over-fragment** (no orphaned figure, no forced cut). This case is required, not just the happy path.
